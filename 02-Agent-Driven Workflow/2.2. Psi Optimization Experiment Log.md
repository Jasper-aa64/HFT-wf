# Psi Optimization Experiment Log

This document is the human-readable trace for the Psi optimization harness. Every evaluator run should also create machine-readable evidence under:

```text
experiments/psi-evaluator-runs/<timestamp>/run.md
experiments/psi_results.tsv
```

The rule from this point forward:

```text
No optimization attempt counts unless it leaves an evidence file and a result row.
```

Schema note:

```text
Rows through 2026-05-07 use the original fixed-threshold result schema.
Rows after the 2.4 gate implementation add gate_class, mean, Welch fields,
perf_gate, perf_reason, and comparison_baseline. Treat older rows as legacy
evidence under the pre-2.4 policy.
```

## 2026-05-07 - Build Gate Diagnosis

Problem:

```text
cmake --build build --target PsiTraderRunner
```

could fail at:

```text
PsiFactorPipline/CMakeFiles/PsiFactorPipline.dir/PsiReadWrite.cpp.obj
```

with no compiler diagnostics.

Root cause:

```text
PATH did not include C:\msys64\ucrt64\bin for MinGW child processes.
```

Evidence:

Same `PsiReadWrite.cpp` compile command:

```text
PATH=/usr/bin:/c/Windows/System32
=> exit=1, stderr 0 bytes

PATH=/ucrt64/bin:/usr/bin:$PATH
=> exit=0, normal warnings
```

Fix:

Added `mingw-relwithdebinfo` preset to `CMakePresets.json`:

```text
PATH=C:/msys64/ucrt64/bin;C:/msys64/usr/bin;$penv{PATH}
CMAKE_CXX_COMPILER=C:/msys64/ucrt64/bin/c++.exe
CMAKE_MAKE_PROGRAM=C:/msys64/ucrt64/bin/mingw32-make.exe
```

Verification:

```powershell
cmake --build --preset mingw-relwithdebinfo --verbose
```

Result:

```text

```
